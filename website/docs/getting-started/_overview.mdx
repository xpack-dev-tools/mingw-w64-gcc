[MinGW-w64 GCC](https://mingw-w64.org) is an advancement of the original mingw.org project, created to support the GCC compiler on Windows systems.

The core of the Mingw-w64 project consists of headers and support libraries that enable GCC-generated output to run on Windows.

**MinGW-w64 GCC** is an open source project hosted on
[SourceForge](https://sourceforge.net/projects/mingw-w64/).
